# 🍰 Menu Receita - Bolo de Cenoura

Um site educativo desenvolvido para fins de aprendizado em desenvolvimento web, apresentando uma receita clássica de bolo de cenoura fofinho com todas as informações necessárias para preparo.

## 📋 Objetivo do Projeto

Este projeto foi criado para fins pedagógicos, permitindo que alunos aprendam conceitos fundamentais de:
- **HTML5** - Estrutura semântica de páginas web
- **Responsividade** - Design mobile-first
- **Navegação** - Estrutura de menus e links internos
- **Organização de conteúdo** - Uso correto de tags semânticas

## 🎯 Funcionalidades

- ✅ Navegação intuitiva com menu principal
- ✅ Seções organizadas: Receita, Ingredientes e Modo de Preparo
- ✅ Lista de ingredientes separada por categorias (Massa e Cobertura)
- ✅ Sugestão de utensílios com imagens e botões de ação
- ✅ Passo a passo do preparo em formato de lista ordenada
- ✅ Layout responsivo para diferentes dispositivos

## 🛠️ Conceitos HTML Explorados

- Semântica HTML5 (`<header>`, `<main>`, `<footer>`, `<section>`, `<fieldset>`)
- Listas aninhadas (listas não-ordenadas e ordenadas)
- Links âncora (`id` e `href`)
- Formulários e elementos de interação (`<button>`)
- Atributos de acessibilidade (`alt` em imagens)

## 👨‍🏫 Como Utilizar em Aulas

1. Use como exemplo de estrutura semântica HTML
2. Estude a organização do conteúdo e hierarquia de títulos
3. Analise o uso de links de navegação interna
4. Implemente CSS para estilizar o layout
5. Torne o site responsivo com media queries
